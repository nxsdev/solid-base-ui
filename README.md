# solid-base-ui

An experimental project that ports the headless UI components from
[Base UI](https://base-ui.com/) to SolidJS 2 Beta.

Upstream reference:

- Website: [base-ui.com](https://base-ui.com/)
- Source: [mui/base-ui](https://github.com/mui/base-ui)

## Status

> [!WARNING]
> This project is still experimental and is not ready for use yet.
> APIs and internal implementation may change significantly, and it should not be treated as a stable library for production or general consumption.
