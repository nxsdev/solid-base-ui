# Core Policy

Manage AI agent rules in this repository with `ruler`.
Do not edit generated files directly. Edit `.ruler/*.md` and `.ruler/ruler.toml`.

When interacting with this repository owner in this workspace, always communicate in Japanese.
Prefer simplicity over excessive precision.
Follow YAGNI, KISS, and DRY.
Do not add backward-compatibility wrappers or fallbacks unless complexity would otherwise increase.
